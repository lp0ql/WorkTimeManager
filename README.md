# WorkTimeManager
作業時間を管理する為のページです。

# [WorkTimeManagerを開く](https://lp0ql.github.io/WorkTimeManager/)

## 時間の構成
- 休憩時間
- 作業時間
- 作業内容を保存したり、整理する時間
- 休憩時間

で時間が構成されています。
作業開始前にも休憩時間があるので、とりあえずこのページを開く事で「もうちょっとしてから作業しよ」から「作業時間になったら作業しよ」に変わります。

## 時間設定の変更
作業時間はローカルストレージの次の値を変更する事で設定できます

- break 休憩時間(秒)
- save 作業内容を保存したり、整理する時間(秒）
- work 作業時間（秒)
